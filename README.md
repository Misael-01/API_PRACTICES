## 📌 Sobre este repositorio
Este espacio es un laboratorio (sandbox) dedicado al diseño, desarrollo y consumo seguro de APIs RESTful. El objetivo de estos mini-proyectos es dominar la arquitectura de microservicios, la integración de endpoints de terceros y el manejo de errores (Try/Except) aplicable a entornos de alta criticidad (Fintech / Cloud).

## 🛠️ Tech Stack Principal
* **Backend:** Python, FastAPI, Flask (opcional).
* **Consumo de APIs:** `requests`, `httpx`.
* **Validación de Datos:** Pydantic.
* **Despliegue:** Docker, Microsoft Azure.

## 📂 Estructura del Repositorio

El laboratorio está dividido en módulos progresivos:


├── 01_FastAPI_Basico/       # Creación de Endpoints (GET, POST, PUT, DELETE)
├── 02_Consumo_Externa/      # Integración con APIs de terceros (Weather, Finanzas)
├── 03_Manejo_Errores/       # Control de excepciones, Timeouts y HTTP Status Codes
├── 04_Auth_Seguridad/       # Implementación de Tokens (JWT) y CORS
└── README.md                # Documentación principal
